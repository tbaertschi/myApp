Einkaufs App

Mehrere Benutzer haben einen Haushalt und können eine Einkaufsliste zusammenstellen.
Es soll eine Auswahl an Lebensmittel geben
Nicht vorhandene Lebensmittel können hinzugefügt werden
Eventuell können zu den einzelnen Lebensmittel Notizen hinzugefügt werden (Menge)

Notification:
Andere Benutzer werden informiert wenn einer die Liste ändert

Login:
Jeder Benutzer benötigt ein Login

Alerts:
Z.Bsp wenn man die ganze Liste löscht
